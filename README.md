# Password-Generator-Bot

# **Installation**

To run the bot locally, follow these steps:

**1) Fork this repository and clone your fork into your machine using**

```bash
git clone git@github.com:USERNAME/Password-Generator-Bot.git
```
Then type this:

```bash
cd Password-Generator-Bot
```


**2) Install dependencies:**

Ensure you have Python installed, then run:

```bash
pip install -r requirements.txt
```

**3) Configuration:**

Create a .env file in the project root and add your Telegram API Token obtained from @BotFather

Below there is an example for your .env file:

```bash
TELEGRAM_BOT_TOKEN=your_token_here
```

**4) Run the bot:**

```bash
python -m bot.main
```

# **Technologies Used**

- Python 3.12.2

- Python-Telegram-Bot

- Python-Dotenv
